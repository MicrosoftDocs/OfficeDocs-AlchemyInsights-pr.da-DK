# <a name="download-and-share-updates"></a>Downloade og dele opdateringer

I stedet for at downloade opdateringer individuelt, kan overførsels optimerings tilstanden bruges til at dele mellem flere pc'er. Hvis du vil have mere at vide, skal du se [leveringsoptimering for Windows 10-opdateringer](https://docs.microsoft.com/windows/deployment/update/waas-delivery-optimization).  

Tilgængelige overførsels tilstande omfatter:  
- Kun HTTP  
- LAN  
- Gruppe  
- Internet  
- Nemmeste  
- Låse
