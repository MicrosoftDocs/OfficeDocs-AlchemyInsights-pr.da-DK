# <a name="put-apps-on-the-desktop"></a>Sæt apps på skrivebordet

Søg efter appen i menuen **Start,** og tag appikonet ved at holde venstre museknap nede og flytte ikonet til skrivebordet.

Du kan også trykke på **Windows-tasten + R** og derefter skrive denne kommando:

`shell:appsfolder`

Dette åbner mappen **Programmer** i **Stifinder**. Du kan nu trække og slippe appikonet til skrivebordet.
