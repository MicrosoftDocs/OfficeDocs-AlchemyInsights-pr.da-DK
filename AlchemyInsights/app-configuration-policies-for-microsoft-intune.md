# <a name="app-configuration-policies-for-microsoft-intune"></a>Politikker for appkonfiguration for Microsoft Intune

Politikker for appkonfiguration i Microsoft Intune indeholder konfigurationsindstillinger for iOS- eller Android-programmer. Disse konfigurationsindstillinger gør det muligt at tilpasse en app. Understøttelse af konfigurationspolitik er begrænset af operativsystemversioner og -programmer.

Du kan finde flere oplysninger [under Politikker for appkonfiguration for Microsoft Intune](https://docs.microsoft.com/intune/app-configuration-policies-overview).
