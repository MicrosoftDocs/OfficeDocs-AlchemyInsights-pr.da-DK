# <a name="app-configuration-policies-for-microsoft-intune"></a>Konfigurations politikker for apps til Microsoft Intune

Konfigurationsindstillinger for appen i Microsoft Intune giver konfigurationsindstillinger for iOS-eller Android-programmer. Disse konfigurationsindstillinger gør det muligt at tilpasse en app. Understøttelse af konfigurationspolitik er begrænset af operativsystemets versioner og programmer.

Du kan få mere at vide under [konfigurations politikker for apps til Microsoft Intune](https://docs.microsoft.com/intune/app-configuration-policies-overview).
