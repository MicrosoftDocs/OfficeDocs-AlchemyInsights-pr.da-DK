# <a name="install-onedrive"></a>Installer OneDrive

OneDrive sync-klient er forudinstalleret med Windows 10. At installere OneDrive til Windows 7 eller Mac, eller at få mobil-app'en til din mobiltelefon eller tablet, gå til [http://onedrive.com/download](http://onedrive.com/download).
