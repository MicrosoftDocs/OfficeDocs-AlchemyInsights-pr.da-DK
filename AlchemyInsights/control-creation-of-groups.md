# <a name="control-creation-of-microsoft-365-groups"></a>Styr oprettelse af Microsoft 365-grupper

Som standard kan alle brugere i virksomheden oprette grupper. Hvis det er nødvendigt, kan du ændre dette og tildele retten til at oprette grupper til en gruppe personer i virksomheden. Bemærk, at dette vil have indflydelse på oprettelse af grupper fra en hvilken som helst Microsoft 365-tjeneste (f.eks Outlook, Teams, Planner og så videre).
  
Vejledning: [Administrer, hvem der kan oprette Microsoft 365-grupper](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
