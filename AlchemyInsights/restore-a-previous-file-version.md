# <a name="restore-a-previous-file-version"></a>Gendanne en tidligere version af filen

Du kan se og gendanne ældre versioner af filerne i OneDrive eller SharePoint med online versionsoversigt. Versionshistorik fungerer med alle filtyper, herunder PDF-filer, CAD-filer, fotos og videoer. Sådan gendannes en tidligere version af filen:
  
1. Åbn SharePoint-dokumentbibliotek eller OneDrive placering.
    
2. Klik til venstre på filen for at markere den, og klik derefter på **versionshistorik**. Hvis du ikke kan se versionsoversigten på kommandolinjen, skal du klikke på flere ( **...** ). 
    
3. Klik på den version, du vil se, eller klik på flere ( **...** ) til højre for en version og derefter klikke på **Gendan**.
