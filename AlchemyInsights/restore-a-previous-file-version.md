# <a name="restore-a-previous-file-version"></a>Gendan en tidligere filversion

Med online Versionshistorik kan du se og gendanne ældre versioner af dine filer i OneDrive eller SharePoint. Versionshistorik fungerer med alle filtyper, herunder PDF'er, CAD-filer, fotos og videoer. Sådan gendannes en tidligere filversion:
  
1. Åbn SharePoint-dokumentbiblioteket eller OneDrive-placeringen.
    
2. Klik til venstre for filen for at markere den, og klik derefter på **versionsoversigt**. Hvis du ikke kan se Versionshistorik på kommandolinjen, skal du klikke på mere ( **...** ). 
    
3. Klik på den version, du vil have vist, eller klik på mere ( **...** ) til højre for en version, og klik derefter på **Gendan**.
