# <a name="learn-how-microsoft-search-in-bing-helps-keep-your-information-secure"></a>Få mere at vide om, hvordan Microsoft Søg i Bing hjælper med at beskytte dine oplysninger

Med Microsoft Søg i Bing:

- Kun brugere, der er logget på, **kan se** søgeresultater.
- Din arbejdsidentitet eller din organisations identitet bruges ikke af Microsoft til at målrette annoncering.
- Din søgeadministrator kan kun se antallet af søgninger efter type, f.eks. efter person eller filtype, og en samlet liste over de mest populære søgninger.
- Det er kun dig, der kan se **søgehistorikken for dit** arbejde.
- Du kan kun finde de arbejdsfiler og oplysninger, der er delt med dig, eller som du har tilladelse til at se.
- Dine oplysninger bliver aldrig tilgængelige for andre organisationer, herunder Microsoft, eller for offentlige søgninger, der udføres på Bing.

Du kan finde flere oplysninger under [Sådan hjælper Microsoft Søg i Bing med at holde dine oplysninger sikre.](https://support.microsoft.com/office/how-microsoft-search-in-bing-helps-keep-your-info-secure-cbce46ae-bb1f-4d0e-86f1-5984f4589113)
