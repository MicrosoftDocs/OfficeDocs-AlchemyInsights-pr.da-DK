# <a name="report-on-file-and-folder-sharing-in-a-sharepoint-site"></a>Rapport om fil- og mappedeling på et SharePoint-websted

Du kan oprette en csv-fil for hver unikke fil, bruger, tilladelse og link på et bestemt SharePoint-websted eller OneDrive-bibliotek. Dette kan hjælpe dig med at forstå, hvordan deling bruges, og om nogen filer eller mapper deles med gæster. For at få flere oplysninger skal du se [Rapport om fil- og mappedeling på et SharePoint-websted](https://docs.microsoft.com/sharepoint/sharing-reports).
