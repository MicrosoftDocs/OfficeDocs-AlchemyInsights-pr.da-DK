# <a name="deploy-microsoft-365-apps"></a>Installere Microsoft 365-apps

Hvis du vil installere Microsoft 365-apps til flere slutbrugere via Office-udrulnings værktøjet, Microsoft Endpoint Configuration Manager, som en del af en diskafbildning af et operativsystem eller i et delt computer scenarie med Windows Server med Fjernskrivebord-tjenester (RDS) eller Terminal Services, kan du overveje at bruge den trinvise vejledning i **Microsoft 365-apps til Enterprise Deployment Advisor**.
  
For at komme i gang skal du sørge for, at du er logget på som lejeradministrator eller Global administrator og derefter klikke her: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance) .
