# <a name="cant-download-files-in-chrome"></a>Kan ikke hente filer i Chrome

**Problem** Vi er bekendt med et problem, der forhindrer nogle kunder i at kunne hente filer, når du bruger Google Chrome webbrowser. 

**Du kan løse problemet** Brug [Microsoft kant](https://www.microsoft.com/en-gb/windows/microsoft-edge) eller en anden webbrowser.
Yderligere oplysninger om at [hente filer og vedhæftede filer Outlook.com](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy).
