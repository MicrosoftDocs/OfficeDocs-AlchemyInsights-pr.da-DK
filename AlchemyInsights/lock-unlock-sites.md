# <a name="lock-and-unlock-sites"></a>Lås og lås op for websteder

Som global Microsoft 365- eller SharePoint-administrator kan du blokere adgangen til et websted eller gøre et websted skrivebeskyttet ved at bruge Microsoft PowerShell til at ændre webstedets låsetilstand. Du kan få mere at vide [under Lås og lås op for websteder](/sharepoint/manage-lock-status).
