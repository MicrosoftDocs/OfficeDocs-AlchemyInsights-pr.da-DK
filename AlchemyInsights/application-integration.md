# <a name="application--integration"></a>Programintegration

- [Integration af Azure Active Directory med programmer Introduktion](https://docs.microsoft.com/azure/active-directory/manage-apps/plan-an-application-integration)  – opsummerer processen til integration af programmer med Azure Active Directory (ad).

- [Hvad er programadministration?](https://docs.microsoft.com/azure/active-directory/manage-apps/what-is-application-management)  -Azure AD skal konfigureres til at integrere med et program. I andre ord skal Azure AD for at finde ud af, hvilke programmer der bruger dem som et identitets system. Processen med at holde Azure-annonce orienteret om disse programmer og af proceduren til at håndtere dem kaldes Programadministration.

- [Selvstudier til integration af programmer i Azure Active Directory](https://docs.microsoft.com/azure/active-directory/saas-apps/tutorial-list)  – for at hjælpe med at integrere dine programmer med Azure ad, har vi udviklet en samling af selvstudier, der fører dig gennem konfigurationen.
