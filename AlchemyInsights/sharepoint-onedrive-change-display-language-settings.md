# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Skift sprog skærmindstillinger i SharePoint eller OneDrive 

<p>Du kan ikke ændre<strong> standardindstillingerne for sprog</strong> for SharePoint- eller OneDrive, fordi disse indstillinger er defineret, da du oprettede abonnementet Office 365. &nbsp;Du kan dog ændre indstillingerne for sprog <strong>vises</strong> .&nbsp;</p> <ol> <li><span style="display: inline !important; float: none; cursor: text; font-family: Verdana,Arial,Helvetica,sans-serif; font-size: 12px; font-style: normal; font-variant: normal; font-weight: 400; letter-spacing: normal; orphans: 2; text-align: left; text-decoration: none; text-indent: 0px; text-transform: none; -webkit-text-stroke-width: 0px; white-space: normal; word-spacing: 0px;">Ændre dine personlige indstillinger</span>, se <a title="ændre dine personlige indstillinger for sprog og område" href="https://support.office.com/en-us/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7">Ændre dine personlige indstillinger for sprog og område</a>.&nbsp;<br /><br /></li> <li>Hvis du vil ændre webstedets brugergrænseflade, se <a title="Vælg de sprog, du vil gøre tilgængelige for et websted brugergrænseflade" href="https://support.office.com/en-us/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8">Vælg de sprog, du vil gøre tilgængelige for et websted-brugergrænsefladen</a>. Finde flere oplysninger om fejlfinding af sprogindstillinger, <a title="du kan ikke ændre standardsproget til en SharePoint Online-websteder" href="https://support.office.com/en-us/article/you-can-t-change-the-default-language-for-a-sharepoint-online-site-collection-40dda07e-6b41-49e9-9828-8805dcb92964">Du kan ikke ændre standardsproget til en SharePoint Online-websteder</a>.</li> </ol> <p>&nbsp;</p>
