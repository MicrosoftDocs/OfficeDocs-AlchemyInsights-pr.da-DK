# <a name="distribution-group-not-showing-in-groups-list"></a>Distributionsgruppe, der ikke vises på listen grupper

Det kan tage op til 60 minutter før den nye distributionsliste vises i midten udveksling admin center.
  
Hvis du vil tilføje medlemmer eller foretage andre ændringer på nuværende tidspunkt, skal du bruge funktionen grupper i [Exchange Admin Center](https://support.office.com/article/https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx).
