# <a name="enable-versioning-for-a-list-or-library"></a>Aktivere versionering for en liste eller et bibliotek

1. Gå til listen eller biblioteket.
    
2. Klik på **Indstillinger**, og klik derefter på **Biblioteksindstillinger** eller **Listeindstillinger**.
    
3. Klik på **Indstillinger for versionering**.
    
4. Vælg de ønskede indstillinger, og klik derefter på **OK**.
