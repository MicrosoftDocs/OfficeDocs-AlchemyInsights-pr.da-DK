# <a name="windows-defender-actions-in-intune"></a>Windows Defender-handlinger i Intune

Intune kan bruges til at udløse en anmodning om scanning og virus signatur i Windows Defender på individuelle enheder.

Når en fjernhandling udløses, afspejles aktiviteten i hændelsesloggen i Windows Defender.

Windows Endpoint Protection-politik gør det muligt at oprette flere indstillinger for Windows Defender-funktioner i Intune og anvendes til sæt af enheder.

Du kan finde flere oplysninger om, hvordan du udløser Windows Defender-handlinger, under [konfigurere og køre on-demand Microsoft Defender Antivirus Scan](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/run-scan-windows-defender-antivirus).
