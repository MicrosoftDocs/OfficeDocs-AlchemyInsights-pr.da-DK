# <a name="windows-defender-actions-in-intune"></a>Windows Defender-handlinger i Intune

Intune kan bruges til at udløse en on-demand scanning og virus signatur opdatering til Windows Defender på individuelle enheder.

Når en fjernhandling er udløst, afspejles aktiviteten i Windows Defender-hændelsesloggen.

Windows Endpoint Protection-politik tillader, at der oprettes yderligere indstillinger for Windows Defender-funktioner i Intune, som anvendes på enhedersæt.

Du kan finde flere oplysninger om udløsere Windows [Defender-handlinger under Konfigurere og køre on-demand Microsoft Defender Antivirus-scanninger](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/run-scan-windows-defender-antivirus).
