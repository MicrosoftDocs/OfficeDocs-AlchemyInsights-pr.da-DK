# <a name="microsoft-365-multi-geo"></a>Microsoft 365 multi-geo

Med Microsoft 365 multi-geo kan organisationen udvide sin Microsoft 365-tilstedeværelse til flere geografiske områder og/eller lande i din eksisterende lejer. Se linkene nedenfor for at få flere oplysninger.

- [Microsoft 365 multi-geo](https://docs.microsoft.com/office365/enterprise/office-365-multi-geo)
- [Plan for Microsoft 365 multi-geo](https://docs.microsoft.com/office365/enterprise/plan-for-multi-geo)
- [Konfigurere multi-geo](https://docs.microsoft.com/office365/enterprise/multi-geo-tenant-configuration)
