# <a name="site-and-page-navigation-in-sharepoint-sites"></a>Websteds- og navigation på SharePoint-websteder

De grundlæggende principper og god praksis for websteds- og navigation, ligeledes gælder for både klassiske og moderne SharePoint-arkitekturer. Men dine muligheder for implementering af navigation er forskellig baseret på rammerne for dine websteder og intranet. For eksempel de nedarvede navigation erfaringer, der er tilgængelige i classic SharePoint-webstedet hierarkier (websteder med underordnede websteder) er ikke tilgængelige i den moderne oplevelse, men [hub websteder](https://support.office.com/article/fe26ae84-14b7-45b6-a6d1-948b3966427f) giver en god måde at opnå navigationsfunktionerne på tværs af websteder tidligere var tilgængelige i administrerede navigations- og hierarkier i klassisk SharePoint.

 Yderligere oplysninger om indstillinger for Navigation, skal du besøge nedenstående links.

- [Tilpas navigation på SharePoint-webstedet](https://support.office.com/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca)

- [Navigationsindstillinger for SharePoint Online](https://docs.microsoft.com/office365/enterprise/navigation-options-for-sharepoint-online)
 
- [Organisere dine SharePoint-websteder med megamenu navigation](https://techcommunity.microsoft.com/t5/Microsoft-SharePoint-Blog/Organize-your-SharePoint-sites-with-megamenu-navigation-and-new/ba-p/328068)
