# <a name="site-and-page-navigation-in-sharepoint-sites"></a>Websteds-og Sidenavigering på SharePoint-websteder

De grundlæggende principper og god praksis for websteds-og sidenavigation gælder også for både klassiske og moderne SharePoint-arkitekturer. Men dine muligheder for at implementere navigation varierer afhængigt af rammerne for dine websteder og intranettet. De nedarvede navigations oplevelser, der er tilgængelige i klassiske SharePoint-websteds hierarkier (websteder med underordnede websteder), er ikke tilgængelige i den moderne oplevelse, men [hubwebsteder](https://support.office.com/article/fe26ae84-14b7-45b6-a6d1-948b3966427f) er en fantastisk måde at opnå navigationsfunktionerne på tværs af websteder på. tidligere tilgængelig i administreret navigation og websteds hierarkier i klassisk SharePoint.

 For mere information om navigationsmuligheder, kan du besøge nedenstående links.

 - [Planlægning af navigation til den moderne SharePoint-oplevelse](https://docs.microsoft.com/sharepoint/plan-navigation-modern-experience)

- [Tilpasse navigationen på SharePoint-webstedet](https://support.office.com/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca)

- [Navigationsindstillinger for SharePoint Online](https://docs.microsoft.com/office365/enterprise/navigation-options-for-sharepoint-online)
 
- [Organiser dine SharePoint-websteder med megamenu-navigation](https://techcommunity.microsoft.com/t5/Microsoft-SharePoint-Blog/Organize-your-SharePoint-sites-with-megamenu-navigation-and-new/ba-p/328068)

- [Administrerede metadata og navigation i SharePoint](https://docs.microsoft.com/sharepoint/dev/general-development/managed-metadata-and-navigation-in-sharepoint)
