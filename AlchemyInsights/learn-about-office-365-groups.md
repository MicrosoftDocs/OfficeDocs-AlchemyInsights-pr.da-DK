# <a name="learn-about-microsoft-365-groups"></a>Få mere at vide om Microsoft 365-grupper

Microsoft 365-grupper er en tjeneste, der fungerer sammen med de Office 365-værktøjer, du bruger allerede, så du kan samarbejde med dine teammedlemmer, når du skriver dokumenter, opretter regneark, arbejder med projektplaner, planlægning af møder eller afsendelse af mail. Du kan finde flere oplysninger i [få mere at vide om Microsoft 365-grupper](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
