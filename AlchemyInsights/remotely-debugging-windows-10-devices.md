# <a name="get-started-with-remotely-debugging-windows-10-devices"></a>Introduktion til fjernfejlfinding af Windows 10-enheder

Sådan foretager du fjernfejlfinding af direkte indhold på en Windows 10-enhed fra en Windows-eller macOS-computer:

1. Konfigurer din Windows 10-enhed til fjernfejlfinding, og opret derefter forbindelse til den fra din udviklings computer.
2. Undersøg og foretag fejlfinding af direkte indhold på din Windows 10-enhed fra din udviklings maskine.
3. Screencast indhold fra din Windows 10-enhed til en DevTools-forekomst på din udviklings maskine.

Hvis du vil have mere at vide, skal du se [Introduktion til fjernfejlfinding af Windows 10-enheder ved hjælp af Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2142172).
