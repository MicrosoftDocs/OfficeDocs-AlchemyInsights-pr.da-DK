# <a name="get-started-with-remotely-debugging-windows-10-devices"></a>Kom i gang med fjerndefinding af Windows 10-enheder

Sådan fjernfejlfinder du liveindhold på en Windows 10-enhed fra en Windows- eller macOS-computer:

1. Konfigurer din Windows 10-enhed til fjernbugging, og opret derefter forbindelse til den fra udviklingscomputeren.
2. Undersøg og fejlfinding af liveindhold på din Windows 10-enhed fra din udviklingsmaskine.
3. Screencast-indhold fra din Windows 10-enhed til en DevTools-forekomst på din udviklingsmaskine.

Du kan få mere at vide [under Kom i gang med fjerndefiner Windows 10-enheder, der bruger Microsoft Edge.](https://go.microsoft.com/fwlink/?linkid=2142172)
