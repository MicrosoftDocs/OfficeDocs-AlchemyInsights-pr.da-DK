# <a name="what-is-microsoft-search-in-bing"></a>Hvad er Microsoft Søg i Bing?

[Microsoft Søg i Bing er](https://docs.microsoft.com/deployoffice/microsoft-search-bing#what-is-microsoft-search-in-bing) en funktion, der hjælper dig med at søge på tværs af din organisation og internettet. Brug Bings adresselinje og søgefelt til at finde dokumenter, interne og eksterne websteder, virksomhedsværktøjer, personer og organisationsdiagrammer, filer, SharePoint-websteder, OneDrive-indhold, Teams og Yammer-samtaler og mange andre ressourcer.
