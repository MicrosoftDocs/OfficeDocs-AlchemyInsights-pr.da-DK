# <a name="workflows-in-sharepoint"></a>Arbejdsgange i SharePoint

<p>SharePoint-arbejdsprocesser er forprogrammerede miniprogrammer, der kan strømline og automatisere en lang række forretningsprocesser. Arbejdsprocesser kan variere fra indsamler signaturer, feedback eller godkendelser for en plan eller et dokument, til sporing af den aktuelle status for en rutinemæssig procedure.</p> <p><strong>Introduktion:</strong> Hvis du&rsquo;re-går i gang med SharePoint-arbejdsgange, se <a href="https://support.office.com/en-us/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3">Introduktion til arbejdsprocesser i SharePoint</a> for at få yderligere oplysninger.</p> <p><strong>Microsoft Flow:</strong> Overvej at bruge Microsoft-Flow i stedet for de klassiske SharePoint-løsninger til arbejdsgange. Se <a href="https://support.office.com/en-us/article/create-a-flow-for-a-list-or-library-in-sharepoint-online-or-onedrive-for-business-a9c3e03b-0654-46af-a254-20252e580d01">oprette et flow til en liste eller et bibliotek i SharePoint Online- eller OneDrive</a>for at få yderligere oplysninger.</p> <p><strong>Arbejdsgang, fejlfinding og fejlretning:</strong> Fejlfinding af arbejdsprocesser, ligner det i tidligere versioner. Yderligere oplysninger, se <a href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/debugging-sharepoint-server-workflows">fejlfinding SharePoint-arbejdsprocesser</a>. Flere oplysninger om fejlfinding finder du <a title="fejlfinding i forbindelse med SharePoint Server workflow valideringsfejl" href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio">Fejlfinding af valideringsfejl i SharePoint Server workflow</a>.&nbsp;</p>
