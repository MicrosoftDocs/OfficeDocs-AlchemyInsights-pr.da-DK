# <a name="use-sensitivity-labels-in-microsoft-365"></a>Brug følsomhedsmærkater i Microsoft 365

Du kan bruge følsomhedsmærkater til at:
- Gennemtving beskyttelsesindstillinger som kryptering eller vandmærker på mærket indhold.
- Beskyt indhold i Office-apps på tværs af forskellige platforme og enheder.
- Beskyt indhold i tredjepartsapps og -tjenester ved hjælp af Microsoft Cloud App Security.
- Udvid følsomhedsmærkater til tredjepartsapps og -tjenester.
- Klassificer indhold uden at bruge nogen beskyttelsesindstillinger.

Du kan finde flere oplysninger i [Få mere at vide om følsomhedsmærkater.](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels)
