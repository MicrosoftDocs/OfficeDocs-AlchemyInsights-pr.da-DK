# <a name="get-started-with-advanced-hunting-queries"></a>Kom i gang med avancerede forespørgselsforespørgsler

Microsoft-sikkerhedseksperter deler [regelmæssigt avancerede forespørgselsforespørgsler i GitHub.](https://go.microsoft.com/fwlink/?linkid=2144624)

Skatteeksperterne tilbyder også avancerede forespørgselsforespørgsler til at finde aktiviteter og indikatorer, der er knyttet [til fremspirende trusler.](https://go.microsoft.com/fwlink/?linkid=2145808) Disse forespørgsler leveres som en del af rapporterne om trusselsanalyse i Microsoft Defender Security Center.
