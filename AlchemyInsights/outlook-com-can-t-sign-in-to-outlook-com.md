# <a name="cant-sign-in-to-outlookcom"></a>Kan ikke logge på Outlook.com

Hvis du har problemer med at logge på din konto, skal du kontrollere din konto ikke har været inaktiv i mere end 365 dage. Efter 365 dage uden aktivitet, e-mailen vil blive slettet og kan ikke gendannes.
  
Din konto kan være midlertidigt blokeret, fordi vi bemærket nogle usædvanlige logonaktivitet. Hvis din konto er spærret, kan du læse hvordan du kan [fjerne blokeringen min konto på Outlook.com](https://support.office.com/article/f4ad2701-d166-4d8b-8a6a-9af2a1f8a4c4).
  
Hvis du har mistanke om din Outlook.com-konto er blevet hacket, kan du se [Mine Outlook.com-konto er blevet hacket](https://support.office.com/article/35993ac5-ac2f-494e-aacb-5232dda453d8).
  
Hvis du har glemt eller mistet adgangskoden, kan du [nulstille din adgangskode](https://go.microsoft.com/fwlink/p/?LinkID=242804).
  
Læs mere om, [Når du ikke kan logge på din Microsoft-konto](https://go.microsoft.com/fwlink/p/?linkid=837479).
