# <a name="pdf-reader-app-in-microsoft-edge"></a>APPEN PDF-læser i Microsoft Edge

Populariteten af PDF-filer fremhæver behovet for en pålidelig, sikker og effektiv PDF-læser, der kan anvendes af virksomhedsvirksomheder. Den funktionsrige PDF-læser indbygget i Microsoft Edge (version 77 og nyere versioner) er netop sådan en app. Det fungerer i Windows og macOS, og det kan bruges til at åbne og anmærke PDF-filer, der er lokale, online eller integreret i websider.

Du kan finde flere oplysninger i [PDF-læseren i Microsoft Edge.](https://docs.microsoft.com/deployedge/microsoft-edge-pdf)
