# <a name="find-results-in-bing-search-by-using-azure-logic-apps"></a>Find resultater i Bing-søgning ved hjælp af Azure Logic-apps

Ved hjælp af Bing-søgning inde fra en logikapp kan du automatisere behandlingen af søgeresultater og gøre disse resultater tilgængelige for andre handlinger. Du kan f.eks. finde nyhedsemner og derefter få Twitter til at sende dem som tweets i din feed.

Du kan få mere at vide [under Find resultater i Bing-søgning ved hjælp af Azure Logic-apps.](https://go.microsoft.com/fwlink/?linkid=2151928)
