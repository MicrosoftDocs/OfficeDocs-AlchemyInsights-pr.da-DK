# <a name="configuring-migration-manager"></a>Konfigurere overførselsstyring

Overførselsstyring centraliserer administrationen af store fildelings overførsler ved at konfigurere en eller flere computere eller virtuelle maskiner (FOS) som overførsel "agenter." For at få mere at vide, kan du se:

- [Konfiguration af klienter til Overførselsstyring](https://docs.microsoft.com/sharepointmigration/mm-setup-clients)

- [Indstillinger for overførselsstyring](https://docs.microsoft.com/sharepointmigration/mm-settings)
