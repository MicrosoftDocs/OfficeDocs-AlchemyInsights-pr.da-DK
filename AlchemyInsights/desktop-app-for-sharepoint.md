# <a name="desktop-app-for-sharepoint"></a>Skrivebordsapp til SharePoint

Din organisation er angivet i det private eksempelprogram for Skrivebordsappen til SharePoint. Microsoft understøtter i øjeblikket ikke Skrivebordsappen til SharePoint (privat prøveversion). Brug dine egne Microsoft private preview-partnere til at få hjælp til Skrivebordsappen til SharePoint.
