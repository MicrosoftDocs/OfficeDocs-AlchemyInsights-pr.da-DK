# <a name="learn-about-external-calendar-sharing-options"></a>Få mere at vide om indstillinger for ekstern kalenderdeling

Hvis du har brug for at planlægge møder med personer uden for organisationen, giver Microsoft 365 dig mulighed for at gøre kalendere tilgængelige for "eksterne brugere", som har brug for at se ledig/optaget tid, men ikke har brugerkonti til dit Microsoft 365-miljø. Yderligere oplysninger finder du i [Dele kalendere med eksterne brugere](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
