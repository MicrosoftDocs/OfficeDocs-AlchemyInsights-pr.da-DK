# <a name="sharepoint-new-features-announced"></a>SharePoint nye funktioner annonceret

På Ignite 2018-konferencen annoncerede Microsoft nye innovationer til SharePoint, herunder nye funktioner til moderne portaler, nye webdele, talrige forbedringer af hubwebsteder, nyheds link, målretning efter målgruppe, tilgængelighed af nye mobilapps, forbindelse med Hold, og mere...
  
Du finder flere oplysninger om disse funktioner i blogindlægget [nye innovationer i SharePoint &amp; Microsoft 365 levere strøm og enkelhed til indholdssamarbejde.](https://go.microsoft.com/fwlink/?linkid=2026502)
  
Din organisation kan tilmelde sig det målrettede udgivelsesprogram for at få tidlig adgang til nye funktioner. Se [konfigurere standard eller målrettede udgivelsesindstillinger i Office 365](https://docs.microsoft.com/office365/admin/manage/release-options-in-office-365).
