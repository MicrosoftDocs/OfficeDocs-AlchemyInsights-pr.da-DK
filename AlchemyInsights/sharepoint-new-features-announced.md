# <a name="sharepoint-new-features-announced"></a>Nyt SharePoint-funktioner, som annonceret

På konferencen Ignite 2018 annoncerede Microsoft nyskabelser til SharePoint, herunder nye funktioner til moderne portaler, nye webdele, hub websteder nyhedshyperlink målretning, og tilgængeligheden af nye mobile applikationer i forbindelse med mange forbedringer Grupper og meget mere...
  
Yderligere oplysninger om disse funktioner finder du i blogindlægget [nyskabelser i SharePoint &amp; Microsoft 365 leverer strøm og enkelhed til indhold samarbejde.](https://go.microsoft.com/fwlink/?linkid=2026502)
  
Organisationen kan deltage i programmet til målrettet Release tidlig adgang til nye funktioner. Se, [hvordan Office 365 kommerciel kunder kan få tidlig adgang til nye funktioner i Office](https://go.microsoft.com/fwlink/?linkid=2026346).
