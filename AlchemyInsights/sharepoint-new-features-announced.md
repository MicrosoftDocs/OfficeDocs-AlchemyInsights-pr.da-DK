# <a name="sharepoint-new-features-announced"></a>Nye SharePoint-funktioner

På Ignite 2018-konferencen annoncerede Microsoft nye innovationer til SharePoint, herunder nye funktioner til moderne portaler, nye webdele, mange forbedringer af hubwebsteder, nyhedslink, målgruppemålretning, tilgængelighed af nye mobilapps, forbindelse med Teams og meget mere...
  
Du kan finde flere oplysninger om disse funktioner i blogindlægget [Nye nyskabelser i SharePoint &amp; Microsoft 365 ved at levere kraft og enkelhed til samarbejde med indhold.](https://go.microsoft.com/fwlink/?linkid=2026502)
  
Din organisation kan deltage i programmet Målrettet frigivelse for at få tidlig adgang til nye funktioner. Se [Konfigurere indstillinger for standardudgivelse eller målrettet udgivelse i Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/manage/release-options-in-office-365).
