# <a name="sharepoint-new-features-announced"></a>Nye SharePoint-funktioner meddelt

På udledningen 2018-konferencen annoncerede vi nye innovationer for SharePoint, herunder nye funktioner til moderne portaler, nye webdele, mange forbedringer til NAV-websteder, nyheds link, målgruppe målretning, tilgængelighed af nye mobilapps, forbindelse med teams og meget mere...
  
Hvis du vil have mere at vide om disse funktioner, kan du se blogindlægget [for nye innovationer i SharePoint &amp; Microsoft 365, der leverer Power og enkelhed til indholds samarbejde.](https://go.microsoft.com/fwlink/?linkid=2026502)
  
Din organisation kan tilmelde sig det målrettede udgivelses program for at få tidlig adgang til nye funktioner. Se [konfigurere standard-eller målrettede udgivelsesindstillinger i Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/manage/release-options-in-office-365).
