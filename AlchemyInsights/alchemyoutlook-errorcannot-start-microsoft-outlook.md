# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Outlook-fejl: "Microsoft Outlook kan ikke startes"

Du har problemer med at starte Outlook 2016, 2013 i Outlook eller Outlook 2010 eller modtagelse "kan ikke starte Microsoft Outlook. Outlook-vinduet kan ikke åbne"fejlmeddelelser?
  
Brug først fejlfindingstrinnene i [denne artikel](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) til at løse følgende startproblemer. 
  
Du kan også køre [værktøj til hukommelsesdiagnosticering avancerede](https://aka.ms/SaRA-OutlookAdvDiagnostics) på den berørte computer.
