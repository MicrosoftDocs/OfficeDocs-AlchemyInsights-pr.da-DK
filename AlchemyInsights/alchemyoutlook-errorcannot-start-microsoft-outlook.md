# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Outlook-fejl: Microsoft Outlook kan ikke startes

Har du problemer med at starte Outlook 2016, Outlook 2013 eller Outlook 2010 eller modtager "Microsoft Outlook kan ikke startes. Kan du ikke åbne Outlook-vinduet "fejlmeddelelser?
  
Du skal først bruge fejlfindingstrinnene i [denne artikel](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) for at få hjælp til at løse disse startproblemer. 
  
Du kan også køre det [avancerede Diagnosticeringsværktøj](https://aka.ms/SaRA-OutlookAdvDiagnostics) på den berørte computer.
