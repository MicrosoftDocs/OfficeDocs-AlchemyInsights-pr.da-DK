# <a name="issues-seeing-calendar-freebusy-information"></a>Problemer med at se oplysninger om ledig/optaget kalender

Hvis du ikke kan se oplysninger om ledig/optaget kalender i andre Outlook-kalendere, eller hvis andre ikke kan se oplysninger om ledig/optaget tid i din kalender, skal du bruge [Microsoft Support and Recovery Assistant til Microsoft 365](https://diagnostics.office.com/).
