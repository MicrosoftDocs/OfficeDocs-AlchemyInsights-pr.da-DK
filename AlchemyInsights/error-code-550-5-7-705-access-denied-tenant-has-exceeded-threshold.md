# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 adgang nægtet, lejer har overskredet grænsen

Denne meddelelse vises, når for meget spam eller masse-mail er blevet sendt af din Office 365-organisation. Hvis du vil fjerne denne blok, skal du forstå og forklare årsagen til en supportmedarbejder, samt løse det underliggende problem.
