# <a name="microsoft-stream-storage"></a>Microsoft Stream-lagring

Microsoft Stream-lejere modtager en fast tildeling på 500 GB lagerplads og yderligere 0,5 GB lagerplads pr. bruger med licens.
Du kan finde flere oplysninger om lagerplads under:

- [Oversigt over Microsoft Stream-storage](https://docs.microsoft.com/stream/license-overview#storage)
- [Føj lagerplads til Microsoft Stream](https://docs.microsoft.com/stream/storage-add-on)
- [Kvoter og begrænsninger i Microsoft Stream](https://docs.microsoft.com/stream/quotas-and-limitations)
