# <a name="enable-seamless-sso"></a>Aktiver sømløs SSO

Azure Active Directory (Azure AD) Seamless Single Sign-On (Seamless SSO) logger automatisk på brugere, når de er på deres stationære pc'er, der er forbundet til virksomhedens netværk.
  
[Følg trinnene i denne artikel for at aktivere Sømløse SSO'er](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
