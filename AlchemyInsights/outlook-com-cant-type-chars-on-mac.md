# <a name="how-to-type-special-characters-on-a-mac"></a>Hvordan du kan indtaste specielle tegn på en Mac

Hvis du vil skrive specialtegn på din Mac, kan du åbne menuen **specialtegn** med tastaturgenvejen ⌘ **kommandoen** + **kontrol** + **plads**.

En hurtig genvej til @ er også **ALT** + **64**.
