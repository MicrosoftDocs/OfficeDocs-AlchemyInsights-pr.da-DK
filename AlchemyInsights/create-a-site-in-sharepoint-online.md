# <a name="create-a-site"></a>Oprette et websted

Du kan oprette forskellige typer websteder i SharePoint Online. Du kan få mere at vide om, hvordan du gør det i det nye SharePoint administration, under [oprette et websted](https://go.microsoft.com/fwlink/?linkid=866295). Du kan finde oplysninger om, hvordan brugere kan oprette deres egne websteder, under [administrere oprettelse af websteder i SharePoint Online](https://go.microsoft.com/fwlink/?linkid=866296).
 
Du kan finde oplysninger om de nye webstedstyper under [gruppere tilknyttede teamwebsteder](https://go.microsoft.com/fwlink/?linkid=866292) eller [Kommunikationswebsteder](https://go.microsoft.com/fwlink/?linkid=866294).
