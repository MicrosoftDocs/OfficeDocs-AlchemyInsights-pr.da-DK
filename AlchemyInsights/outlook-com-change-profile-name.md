# <a name="change-your-profile-name"></a>Ændre profilnavnet på din

1. Gå til [din Microsoft-kontooplysninger](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. Vælg **Rediger navn**under dit navn.
3. Angiv et fornavn og efternavn, og vælg **Gem**.

Der kan gå op til 24 timer efter ændringer i dit profilnavn kan træde i kraft.
  
Hvis du vil vide mere, se [ændre dit profilbillede eller navn i Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).
