# <a name="block-pop-up-windows-in-microsoft-edge"></a>Bloker pop op-vinduer i Microsoft Edge

1. Gå til **... (Indstillinger med mere)**  >  **Indstillinger**  >  **Webstedstilladelser.**
2. Vælg **Pop op-vindue og omdirigeringer.**
3. Slå **Bloker til.**
