# <a name="issues-while-migrating-data-to-sharepoint-online"></a>Problemer mens data overflyttes til SharePoint Online

Det vises, du vil overføre mere end 100TB data til SharePoint Online. Følg nedenstående trin, så vi kan hjælpe dig så hurtigt som muligt. 

1. Vælg **Anmod om ny tjeneste**, og derefter **anmode om ny tjeneste**. 
2. Lad den titel og beskrivelse som **SharePoint migration over 100 TB**.
3. Når billetten er blevet sendt, skal du opdatere den med følgende oplysninger: 

    - Anslåede størrelse af din migrering.
    - En beregning af, hvor du vil starte og fuldføre overførslen.
    - Beskriver, hvor du overfører dit indhold fra, som SharePoint Server, boks, GDrive, filshares, osv...
