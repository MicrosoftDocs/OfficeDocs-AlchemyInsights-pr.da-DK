# <a name="issues-while-migrating-data-to-sharepoint-online"></a>Problemer mens data overflyttes til SharePoint Online

Det vises, du vil overføre mere end 100TB data til SharePoint Online.

Klik på **Ny Service anmodning** pr. telefon/e-mail nedenfor, og lad forespørgslen som SharePoint migration over 100 TB i tekstboksen.

Når billetten er blevet sendt, skal du opdatere den med følgende oplysninger: 

- Virksomhedens navn og URL-adressen til lejeradministration
- Anslåede størrelse af din migrering
- Et skøn over, hvor du vil starte og afslutte din migrering
- Beskriver, hvor du overfører dit indhold fra, som SharePoint Server, boks, GDrive, filshares, osv.
