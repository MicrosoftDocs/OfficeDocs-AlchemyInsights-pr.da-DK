# <a name="changes-to-external-user-access"></a>Ændringer af ekstern bruger adgang

Fra og med 23. marts 2018 vil eksterne brugere ikke længere kunne få adgang til indhold, der er delt med alle, alle godkendte brugere eller alle formularer-brugere, selvom de er blevet inviteret eller føjet til autoriserede grupper. Eksterne brugere kan kun få adgang til indhold, der er blevet delt med brugeren eller med grupper, som brugeren tilhører. Hvis du vil beholde dine eksisterende indstillinger, skal du se: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
