# <a name="error-onedrive-cannot-connect-to-windows"></a>Fejl: OneDrive kan ikke oprette forbindelse til Windows

Hvis du vil løse fejlmeddelelsen, skal du søge efter den cldflt.sys fil for at se, om den kører, og hvis den ikke er, skal du aktivere den. 
  
Hvis du vil have mere at vide om, hvordan du aktiverer cldflt.sys-filen, skal du se [OneDrive kan ikke oprette forbindelse til Windows ", når du får adgang til filer i OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
