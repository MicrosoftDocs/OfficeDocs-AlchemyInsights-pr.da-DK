# <a name="access-services"></a>Access Services

Funktionen Access Services i SharePoint Online, er der udgået. Nye Access-baserede web-applikationer og webdatabaser kan ikke længere oprettes, og eksisterende web-apps og webdatabaser lukkes ved April, 2018. Stationære Access-databaser (.accdb) er ikke berørt af ændringen. Vi anbefaler Microsoft PowerApps som et alternativ til Access web apps og webdatabaser. 
  
[Få mere at vide om Microsoft PowerApps](https://powerapps.microsoft.com/)
