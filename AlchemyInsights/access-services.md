# <a name="access-services"></a>Få adgang til tjenester

Funktionen Access-tjenester i SharePoint Online udgår. Der kan ikke længere oprettes nye Access-baserede webapps og webdatabaser, og eksisterende webapps og webdatabaser lukkes i april 2018. Access Desktop-databaser (.accdb) påvirkes ikke af denne ændring. Vi anbefaler Microsoft PowerApps som et alternativ til Access-webapps og webdatabaser. 
  
[Få mere at vide om Microsoft PowerApps](https://powerapps.microsoft.com/)
