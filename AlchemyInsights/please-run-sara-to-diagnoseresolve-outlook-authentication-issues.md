# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Du kan bruge Pia til at diagnosticere og løse problemer med Outlook godkendelse

Det anbefales, at du bruger den [Outlook holder bede om adgangskoden](https://aka.ms/SaRA-OutlookPwdPrompt-Alchemy) diagnostiske på den berørte computer til fejlfinding hvor hele tiden bedt om en adgangskode. Denne [Pia](https://diagnostics.office.com/#/) diagnostiske understøtter automatiseret kontrol og returnerer mulige løsninger, som du kan bruge til at løse eventuelle registrerede problemer.
