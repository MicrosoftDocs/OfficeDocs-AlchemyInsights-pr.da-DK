# <a name="upload-process-overview-in-microsoft-stream"></a>Oversigt over overførselsproces i Microsoft Stream

Når du uploader en video, koder Microsoft Stream den automatisk i flere bithastigheder, så videoen kan streames ved hjælp af adaptiv bitratestreaming. Microsoft Stream optimerer bithastigheden og kvaliteten af afspilningsvideoen baseret på seerens tilgængelige båndbredde.

- [Oversigt over overførselsproces i Microsoft Stream](https://docs.microsoft.com/stream/upload-process-overview)
