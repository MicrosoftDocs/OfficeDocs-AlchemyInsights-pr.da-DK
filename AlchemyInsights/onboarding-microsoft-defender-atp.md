# <a name="onboarding-microsoft-defender-atp"></a>Onboarding Microsoft Defender ATP

Du kan finde ressourcer til at indbygge Microsoft Defender ATP til Windows 10, Windows-servere og Mac- og Linux-maskiner under: 

- **Windows 10-maskiner** Afhængigt af virksomhedens behov kan du vælge mellem indstillinger til ombord på en maskine, der kører Windows 10. Du kan finde flere oplysninger [under Onboardingværktøjer og -metoder til Windows 10-enheder](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-endpoints). 

- **Windows-servere** Hvis du vil være indbygget i Microsoft Defender ATP til din version af Windows Server, skal du se [Indbyggede servere til Microsoft Defender ATP-tjenesten](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-server-endpoints).

- **Slutpunkter,** der ikke er windows  Før du går om bord på et system, der ikke er Windows, skal du opfylde systemkravene til [Mac](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-atp-mac#system-requirements) eller [Linux](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-atp-linux#system-requirements).

    Når du har bekræftet systemkravene, skal du se [Onboard ikke-Windows-enheder](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-endpoints-non-windows#onboarding-non-windows-machines).
