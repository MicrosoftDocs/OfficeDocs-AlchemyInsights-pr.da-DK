# <a name="onboarding-microsoft-defender-atp"></a>Indbygget Microsoft Defender ATP

Hvis du har en indbygget Microsoft Defender-DTT til Windows 10, Windows Server og Mac og Linux-computere, skal du se: 

- **Windows 10-computere** Afhængigt af virksomhedens behov kan du vælge mellem indstillinger, der er indbygget i en maskine, der kører Windows 10. Hvis du vil have mere at vide, skal du se [indbyggede værktøjer og metoder til Windows 10-enheder](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-endpoints). 

- **Windows-servere** Hvis du vil have indbygget Microsoft Defender-DTT til din version af Windows Server, skal du se [integrerede servere til Microsoft Defender DTT-tjenesten](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-server-endpoints).

- **Ikke-Windows-slutpunkter**  Før du onboarder et ikke-Windows-system, skal du opfylde systemkravene til [Mac](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-atp-mac#system-requirements) eller [Linux](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-atp-linux#system-requirements).

    Se [integrerede ikke-Windows-enheder](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-endpoints-non-windows#onboarding-non-windows-machines), når du har bekræftet systemkravene.
