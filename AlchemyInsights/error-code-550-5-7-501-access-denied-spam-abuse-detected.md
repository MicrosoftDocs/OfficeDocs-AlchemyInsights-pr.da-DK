# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 Adgang nægtet, spam misbrug opdaget

Denne meddelelse vises typisk, når brugerne sender mails fra IP-adresser ved hjælp af det oprindelige *.onmicrosoft.com-domæne,* der er tildelt nye lejere i Microsoft 365. Den nemmeste måde at løse dette problem på er at:

1. [Føj et domæne til din lejer](https://docs.microsoft.com//office365/admin/setup/add-domain).

2. [Skift dine brugeres primære mailadresse](https://docs.microsoft.com//office365/admin/add-users/change-a-user-name-and-email-address) til det nye brugerdefinerede domæne, du lige har tilføjet.
