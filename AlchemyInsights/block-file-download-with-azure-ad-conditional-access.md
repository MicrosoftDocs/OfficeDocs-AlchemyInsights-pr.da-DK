# <a name="block-file-download-with-azure-ad-conditional-access"></a>Blokere filoverførsel med betinget adgang til Azure AD

Politikker for cloud-appsikkerhed giver dig mulighed for at begrænse en session baseret på enhedstilstand. Hvis du vil udføre styringen af en session ved hjælp af enheden som betingelse, skal du oprette både en politik for betinget adgang og en sessionspolitik.

- [Bloker filoverførsel med betinget adgang til Azure AD](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)
