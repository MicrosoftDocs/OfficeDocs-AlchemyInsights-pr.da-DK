# <a name="proxyaddress-incorrect"></a>ProxyAddress er forkert

Når et objekt synkroniseres med Azure AD, sammenlignes de værdier, der er angivet i attributten proxyAddresses i Active Directory, med Azure AD-regler, og derefter udfyldes attributten proxyAddresses i Azure AD. Derfor er værdierne i attributten proxyAddresses for objektet i Active Directory muligvis ikke de samme som værdierne i proxyAddresses-attributten i Azure AD.
  
Hvis du vil have mere at vide om, hvordan proxyadressen udfyldes, skal du se [Sådan udfyldes attributten proxyAddress i Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
