# <a name="proxyaddress-incorrect"></a>ProxyAddress forkert

Når et objekt synkroniseres til Azure AD, sammenlignes de værdier, der er angivet i proxyAddresses-attributten i Active Directory, med Azure AD-regler, og derefter udfyldes proxyAddresses-attributten i Azure AD. Derfor kan værdierne for proxyAddresses-attributten for objektet i Active Directory ikke være de samme som værdierne for attributten proxyAddresses i Azure AD.
  
Hvis du vil vide mere om, hvordan proxyAddress er udfyldt, kan du se, [hvordan proxyAddress-attributten udfyldes i Azure ad](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
