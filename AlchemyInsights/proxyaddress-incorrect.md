# <a name="proxyaddress-incorrect"></a>ProxyAddress er forkert

Når et objekt synkroniseres med Azure AD, sammenlignes de værdier, der er angivet i proxyAddresses-attributten i Active Directory, med Azure AD-regler, og derefter udfyldes attributten proxyAddresses i Azure AD. Derfor er værdierne for attributten proxyAddresses for objektet i Active Directory muligvis ikke de samme som værdierne for attributten proxyAddresses i Azure AD.
  
Hvis du vil vide mere om, hvordan proxyadressen er udfyldt, kan du se, [hvordan attributten-attributten er udfyldt i Azure ad](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
