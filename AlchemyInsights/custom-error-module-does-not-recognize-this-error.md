# <a name="getting-custom-error-module-does-not-recognize-this-error-when-downloading-a-message-trace"></a>Hentning af "brugerdefineret fejlmodul genkender ikke fejlen" ved hentning af en meddelelsessporing?

Vi er opmærksomme på problemet og arbejder på en rettelse.  I mellemtiden kan du prøve at **fjerne din browsercache**eller bruge InPrivate-tilstand til at løse problemet.
