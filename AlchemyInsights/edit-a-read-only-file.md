# <a name="edit-a-read-only-file"></a>Redigere en skrivebeskyttet fil

Når du klikker på et dokuments navn i dokumentbiblioteker, kan Office-programmet uventet åbne det som en skrivebeskyttet fil. Hvis du vil åbne den til redigering, skal du klikke på **Åbn** (de tre prikker) og derefter klikke på **Rediger.**
