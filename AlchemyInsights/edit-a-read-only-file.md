# <a name="edit-a-read-only-file"></a>Redigere en skrivebeskyttet fil

I dokumentbiblioteker, når du klikker på et dokumentnavn, Office-programmet muligvis uventet åbne den som en skrivebeskyttet fil. Klik på **Åbn** (tre prikker) for at åbne den til redigering, og klik derefter på **Rediger**
