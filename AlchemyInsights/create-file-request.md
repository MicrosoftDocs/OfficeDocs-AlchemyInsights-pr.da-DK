# <a name="how-to-create-a-file-request"></a>Sådan oprettes en filanmodning

Filanmodnings funktionen i OneDrive giver dig mulighed for at vælge en mappe, hvor andre kan uploade filer ved hjælp af et link, som du sender dem. De brugere, du anmoder om filer fra, kan ikke se, hvad der er i mappen-de kan kun uploade filer til den.

[Oprette en filanmodning](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af)
