# <a name="how-to-create-a-file-request"></a>Sådan oprettes en filanmodning

Med funktionen til filanmodning i OneDrive kan du vælge en mappe, hvor andre kan uploade filer ved hjælp af et link, du sender dem. De brugere, du anmoder om filer fra, kan ikke se, hvad der er i mappen – de kan kun overføre filer til den.

**Bemærk!** En administrator skal aktivere [Alle links i](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off) OneDrive, for at filanmodninger kan fungere.

- Du kan få mere at vide [under Opret en filanmodning](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af).
