# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Understøttet version af SharePoint-overførselsværktøjet



I øjeblikket understøtter SharePoint-overførselsværktøjet kun overførsler fra SharePoint 2013. Versioner 2010 og 2016 understøttes ikke på nuværende tidspunkt.
  
Yderligere oplysninger finder [du i Introducing the SharePoint Migration Tool](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
