# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Understøttet version af SharePoint-overflytningsværktøjet



I øjeblikket understøtter SharePoint-overflytningsværktøjet kun overførsler fra SharePoint 2013. Version 2010 og 2016 understøttes ikke på dette tidspunkt.
  
Du finder flere oplysninger i [Introduktion til SharePoint-overflytningsværktøjet](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
