# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Understøttet version af SharePoint-overførselsværktøjet



SharePoint-overførselsværktøjet understøtter i øjeblikket kun overførsler fra SharePoint 2013. Version 2010 og 2016 understøttes ikke på nuværende tidspunkt.
  
Du kan finde flere oplysninger i [Introduktion til SharePoint-overførselsværktøjet](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
