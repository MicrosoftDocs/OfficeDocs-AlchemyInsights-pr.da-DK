# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Understøttet version af SharePoint-overflytningsværktøjet



SharePoint-overflytningsværktøjet understøtter i øjeblikket kun overførsler fra SharePoint 2013. 2010 versioner og 2016 understøttes ikke på nuværende tidspunkt.
  
Yderligere oplysninger finder du under [Introduktion til SharePoint-overflytningsværktøjet](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
