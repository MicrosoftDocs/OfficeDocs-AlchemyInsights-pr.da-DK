# <a name="change-your-language-date-and-time-format-or-time-zone"></a>Ændre sprog, dato og format for klokkeslæt eller tidszone

1. Vælg **Indstillinger**øverst på siden, > **få vist alle Outlook-indstillinger**.
2. Vælg **Generelt** > **sprog og tid**.
3. Vælg det sprog, du vil bruge.
4. Vælg **Gem**.

Hvis du vil vide mere, kan du se [ændre tidszonen og sprogindstillingerne i Outlook på internettet](https://support.office.com/article/65239869-12e7-4a9d-bca1-76b0ad7ce273).
