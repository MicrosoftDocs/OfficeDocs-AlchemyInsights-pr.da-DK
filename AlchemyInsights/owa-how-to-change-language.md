# <a name="change-language"></a>Skift sprog

1.    Logge på Outlook på internettet.
2. Vælg **Indstillinger**øverst på siden, > **få vist alle Outlook-indstillinger**.
3. Vælg **Generelt** > **sprog og tid**.
4. Vælg det sprog, du vil bruge.
5. Vælg **Gem**.
 
Hvis du vil vide mere, kan du se [ændre tidszonen og sprogindstillingerne i Outlook på internettet](https://support.office.com/article/65239869-12e7-4a9d-bca1-76b0ad7ce273).
