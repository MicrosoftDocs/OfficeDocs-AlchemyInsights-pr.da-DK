# <a name="cant-delete-files-or-photos-from-files"></a>Kan ikke slette filer eller billeder fra filer

Filer vises, filer og fotos, der er vedhæftet meddelelser i postkassen. Du kan ikke slette filer eller fotos direkte fra filer. For at slette filer og billeder, skal du slette de e-mail-meddelelser, som de er knyttet til:

Yderligere oplysninger, Læs, [hvordan du sletter filer eller billeder i Outlook.com](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx).
