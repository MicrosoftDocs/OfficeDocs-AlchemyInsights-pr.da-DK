# <a name="about-universal-print"></a>Om universel udskrift

Universal Print er inkluderet i erhvervs- og uddannelsesabonnementer på Microsoft 365 og Windows 10, men kan også købes som et enkeltstående abonnement.

Få mere at vide om, hvordan du konfigurerer printere, giver brugerne adgang til og fuldfører dit første udskriftsjob på siden Onboarding til Universal Print.

Du kan finde flere oplysninger [i License Universal Print | Microsoft Docs.](https://docs.microsoft.com/universal-print/fundamentals/universal-print-license)
