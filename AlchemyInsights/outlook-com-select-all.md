# <a name="how-to-select-all-in-outlookcom"></a>Sådan "Markér alt" i Outlook.com

Du kan kun vælge alle, hvis alle meddelelser er synlige. Dette sker for at forhindre utilsigtede handlinger for meddelelser, du ikke kan se.

**CTRL + klik** markerer flere individuelle meddelelser.

**Skift + klik** markerer alle meddelelser mellem den første og den sidste meddelelse, du vælger.
