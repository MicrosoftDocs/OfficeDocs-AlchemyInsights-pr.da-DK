# <a name="screen-sharing-with-skype-consumer-users"></a>Skærmdeling med Skype-forbrugerbrugere

Du kan finde generelle instruktioner om, hvordan du deler dit program, hvis du vil vide mere om begrænsninger i forbindelse med deling og fejlfindingsressourcer, ved at se Del din skærm [i Skype for Business.](https://support.microsoft.com/office/share-and-present-content-from-skype-meetings-app-skype-for-business-web-app-234b0c06-a88d-4707-904c-4fd6c571fc01)  

Skype for Business-brugere kan i øjeblikket ikke dele skærmen med Skype-forbrugerbrugere. Klik [her for at](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) give feedback vedrørende aktivering af skærmdeling med Skype-forbrugerbrugere.
