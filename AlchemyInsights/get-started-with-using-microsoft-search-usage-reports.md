# <a name="get-started-with-using-microsoft-search-usage-reports"></a>Kom i gang med at bruge brugsrapporter fra Microsoft Søg

Brugsrapporter for Microsoft Søg giver indsigt i, hvordan søgning fungerer i organisationen. Denne indsigt kan hjælpe dig med at finde indhold og gøre søgning til en mere nyttig og praktisk oplevelse for brugerne i din organisation.

Du kan få mere at vide i [Microsoft-søgeanvendelsesrapporter.](https://go.microsoft.com/fwlink/?linkid=2152048)
