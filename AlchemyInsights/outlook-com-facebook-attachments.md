# <a name="unable-to-attach-or-upload-files-from-facebook"></a>Kan ikke vedhæfte eller uploade filer fra Facebook

På grund af en ændring på Facebook-vilkår og-betingelser, kan du efter oktober 23 2019 ikke længere overføre vedhæftede filer direkte fra Facebook, og Facebook vil ikke længere blive vist på listen over dine Cloud-placeringer. 

Du kan administrere din liste over andre Cloud-placeringer på dine [lagerkonti](https://go.microsoft.com/fwlink/?linkid=2111075).

Få mere at vide om [lagerkonti](https://support.office.com/article/477cb7cc-5732-4c40-8f23-30472de8138a).
