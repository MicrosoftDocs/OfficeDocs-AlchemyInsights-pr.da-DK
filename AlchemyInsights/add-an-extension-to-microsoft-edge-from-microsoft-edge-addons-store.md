# <a name="add-an-extension-to-microsoft-edge-from-the-microsoft-edge-add-ons-store"></a>Tilføj en udvidelse til Microsoft Edge fra tilføjelsesprogram Microsoft Edge-tilføjelsesprogrammer

Sådan føjer du en udvidelse til Microsoft Edge fra tilføjelsesprogram Microsoft Edge-tilføjelsesprogrammer:

- Åbn Microsoft Edge.
- Vælg **... >-udvidelser > [få udvidelser til Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2136408)**.
- Vælg den udvidelse, du gerne vil tilføje, og vælg derefter **Hent**.
- I bekræftelsesdialogboksen skal du nøje gennemgå de ændringer, som udvidelsen kan foretage, og vælge **Tilføj udvidelse**.

Du får derefter en bekræftelse på, at udvidelsen er blevet tilføjet.
