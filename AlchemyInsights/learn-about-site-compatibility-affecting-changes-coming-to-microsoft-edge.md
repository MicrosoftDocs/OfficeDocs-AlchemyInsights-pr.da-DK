# <a name="learn-about-site-compatibility-affecting-changes-coming-to-microsoft-edge"></a>Få mere at vide om webstedskompatibilitet, der påvirker ændringer, der kommer til Microsoft Edge

Internettet udvikler sig konstant for at forbedre brugeroplevelsen, sikkerheden og beskyttelsen af personlige oplysninger. Nogle ændringer er betydelige nok til at påvirke funktionaliteten af eksisterende sider. Microsoft Edge-teamet opdaterer siderne, efterhånden som overvejelserne udvikler sig, tidslinjer bliver udfyldt, og [ændringer meddeles.](https://go.microsoft.com/fwlink/?linkid=2135534)
