# <a name="control-who-creates-microsoft-365-groups-and-microsoft-teams"></a>Styre, hvem der opretter Microsoft 365-grupper og Microsoft Teams

Som standard kan alle brugere i organisationen oprette Microsoft 365-grupper og Microsoft Teams. Du kan ændre standarden og tildele retten til at oprette Microsoft 365-grupper eller Teams til et undersæt af brugere i organisationen. Denne ændring påvirker også oprettelse af grupper fra alle Microsoft 365-tjenester, f.eks.

Du kan finde flere oplysninger under [Administrere, hvem der kan oprette Microsoft 365-grupper](https://support.office.com/article/Manage-who-can-create-Office-365-Groups-4c46c8cb-17d0-44b5-9776-005fced8e618) og [Microsoft Teams](https://aka.ms/rtsf).
