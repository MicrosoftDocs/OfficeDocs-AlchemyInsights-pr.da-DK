# <a name="email-forwarding-report"></a>Rapport om videresendelse af mail

Hvis du vil se en rapport om, hvor mange automatisk videresendte meddelelser der er i din organisation, skal du se Indsigt i auto-videresendte meddelelser [i Security & Compliance Center](https://docs.microsoft.com/microsoft-365/security/office-365-security/mfi-auto-forwarded-messages-report) i Microsoft [365 Security &amp; Compliance Center.](https://protection.office.com/#/homepage)
  
Bemærk! Hvis det er første gang, du logger på Security &amp; Compliance Center, skal du muligvis opdatere browseren efter et par minutter for at se alle de tilgængelige indstillinger.
