# <a name="setting-up-project"></a>Konfiguration af Project

 Hvis du vil konfigurere Microsoft Project, skal du følge trinnene i [denne artikel](https://support.office.com/article/7059249b-d9fe-4d61-ab96-5c5bf435f281.aspx).

**Meddelelse til Mac-brugere:** Projekt produktet er ikke tilgængeligt for MacOS på nuværende tidspunkt. 
  
Hvis du kører fejlmeddelelser under installationen, skal du [installere Project ved hjælp af support-og genoprettelses assistent](https://aka.ms/SaRA-ProjectSetupScenario).
