# <a name="setting-up-project"></a>Opsætning af projekt

 Hvis du vil konfigurere Microsoft Project, skal du følge trinnene i [denne artikel](https://support.office.com/article/7059249b-d9fe-4d61-ab96-5c5bf435f281.aspx).

**Meddelelse til Mac-brugere:** Project-produktet er ikke tilgængeligt til MacOS på nuværende tidspunkt. 
  
Hvis du løber ind i fejl under installationen, [skal du installere Project ved hjælp af Support- og Genoprettelsesassistent](https://aka.ms/SaRA-ProjectSetupScenario).
