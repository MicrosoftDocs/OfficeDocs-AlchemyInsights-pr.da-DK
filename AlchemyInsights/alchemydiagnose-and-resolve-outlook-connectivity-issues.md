# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>Diagnosticere og løse problemer med Outlook-forbindelsen

Hvis du vil diagnosticere og løse dine problemer med Outlook-forbindelsen, kan du køre [Outlook-forbindelsesværktøjet](https://aka.ms/SaRA-OutlookDisconnect-Alchemy) på den berørte maskine.
  
Dette værktøj udfører diagnosticeringskontroller og foreslår mulige løsninger, som du kan bruge til at løse outlook-forbindelsesproblemet.
