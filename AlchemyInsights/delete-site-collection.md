# <a name="delete-a-sharepoint-site-collection"></a>Slet SharePoint-websteder

1. I den klassiske SharePoint admin center, på siden websteder, ved at markere afkrydsningsfeltet ud for den gruppe af websteder.
    
2. Vælg fanen grupper af websteder **slettes**.
