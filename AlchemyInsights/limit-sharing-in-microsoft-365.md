# <a name="limit-sharing-in-microsoft-365"></a>Begræns deling i Microsoft 365

Selvom du ikke kan deaktivere intern deling helt eller fjerne knappen Del fra websteder, er der en række forskellige måder, hvorpå du kan begrænse deling i Microsoft 365 for at opfylde organisationens behov. Yderligere oplysninger finder du under [Begrænse deling i Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).
