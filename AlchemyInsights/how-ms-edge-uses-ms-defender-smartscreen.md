# <a name="how-microsoft-edge-uses-microsoft-defender-smartscreen"></a>Sådan bruger Microsoft Edge Microsoft Defender SmartScreen

For at sikre dig, at du er sikker på internettet, bruger Microsoft Edge Microsoft Defender SmartScreen som et tidligt advarsels system mod websteder, der perpetrate phishing-angreb og distribuerer malware via fokuseret angreb.

Administratorer kan konfigurere Microsoft Defender SmartScreen med Gruppepolitik-, Microsoft Intune-eller MDM-indstillinger (Mobile Device Management). Afhængigt af hvordan du konfigurerer Microsoft Defender SmartScreen, kan du vise brugere en advarsels side og lade dem fortsætte til webstedet eller blokere webstedet fuldstændigt.

Hvis du vil have mere at vide, skal du se [Microsoft Edge support til Microsoft Defender SmartScreen](https://go.microsoft.com/fwlink/?linkid=2133081) og [Microsoft Defender Application Guard](https://go.microsoft.com/fwlink/?linkid=2132839).
