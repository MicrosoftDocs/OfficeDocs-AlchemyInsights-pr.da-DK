# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Microsoft 365-URL-adresser og IP-adresseområder

Microsoft 365 kræver forbindelse til internettet via bestemte IP-adresser, der kaldes *slutpunkter*.
Hvis du vil have en liste over slutpunkter, der skal være tilgængelige, skal du se [Microsoft 365-URL-adresser og IP-adresseområder](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

Microsoft 365-IP-adressen og URL-adressen til webtjenesten hjælper dig med bedre at identificere og differentiere Microsoft 365-netværkstrafikken, hvilket gør det nemmere for dig at evaluere, konfigurere og holde dig opdateret med ændringer. Se [Microsoft 365 IP address and URL-adresse til webtjeneste](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service).
