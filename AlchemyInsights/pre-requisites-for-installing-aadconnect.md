# <a name="pre-requisites-for-installing-aadconnect"></a>Forudsætninger for installation af AADConnect

Azure AD Connect skal installeres på Windows Server 2012 eller nyere. Denne server skal være tilknyttet et domæne, og det kan være domænecontroller eller en medlemsserver.  Hvis du vil have en komplet liste over Azure AD Connect-krav og -forudsætninger, skal du gennemgå [Forudsætninger for Azure AD Connect](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-install-prerequisites).
