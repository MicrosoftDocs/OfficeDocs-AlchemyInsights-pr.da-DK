# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>Onboard en Windows 10-enhed til Microsoft Defender Advanced Threat Protection (MDATP)

MDATP understøtter følgende installationsværktøjer og -metoder til Windows 10:

- Gruppepolitik
- Microsoft Endpoint Configuration Manager
- Administration af mobilenheder, herunder Microsoft Intune
- Lokale scripts

Du kan få mere at vide [under Onboarding-værktøjer og -metoder til Windows 10-enheder.](https://go.microsoft.com/fwlink/?linkid=2143460)
