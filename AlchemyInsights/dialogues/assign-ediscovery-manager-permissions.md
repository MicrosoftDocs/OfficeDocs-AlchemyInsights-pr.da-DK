# <a name="assign-ediscovery-manager-permissions"></a>Tildel eDiscovery Manager-tilladelser

Udfør følgende trin for at tildele eDiscovery Manager-tilladelser:

1. I [Office 365 Security & Compliance Center](https://sip.protection.office.com/)skal du vælge Tilladelser > **eDiscovery Manager.**
2. Klik på Rediger ud for eDiscovery Manager **i** **ruden eDiscovery Manager.** 
3. Klik på **Vælg eDiscovery Manager** i ruden Redigeringsvælger. 
4. I **ruden Vælg eDiscovery Manager** skal du **klikke på +** Tilføj og derefter vælge brugernes navne.
5. Vælg **Tilføj > færdig > Gem.**
