# <a name="changing-public-folder-permissions"></a>Ændring af tilladelser til offentlige mapper

Tilladelser til offentlige mapper kan ændres af brugere og administratorer i Outlook. Administratorer kan også styre tilladelser fra Exchange Admin Center (EAC), ved at gøre følgende:
  
1. IIn Microsoft 365 admin center, gå til **Admin Centre** \> **Exchange**.

2. Vælg **offentlige mapper**.

3. Herfra kan du ændre tilladelserne for individuelle offentlige mapper ved at tildele tilladelser sikkerhedsgrupper. Brugeren skal have ejerrettigheder til mappen for slutbrugeren at ændre tilladelser til offentlige mapper.
