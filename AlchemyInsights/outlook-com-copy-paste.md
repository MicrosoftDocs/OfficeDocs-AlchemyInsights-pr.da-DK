# <a name="copy-and-paste-in-outlookcom"></a>Kopiere og indsætte i Outlook.com

Der er ikke kopier eller Indsæt knapper i Outlook.com, men du kan bruge tastaturgenveje:

**På en PC:** CTRL + C for at kopiere, CTRL + X for at klippe og CTRL + V for at indsætte.

**På en Mac:** CMD + C for at kopiere, CMD + X for at klippe og CMD + V for at indsætte.
