# <a name="using-self-service-sign-up-in-your-organization"></a>Brug af selvbetjeningstilmelding i din organisation

Med selvbetjeningstilmelding kan brugere tilmelde sig til at bruge tjenester, der betales af dit abonnement.

- Hvis du vil styre, om brugerne kan tilmelde sig selvbetjeningsabonnementer, skal du bruge [Cmdlet'en Set-MsolCompanySettings](https://docs.microsoft.com/powershell/module/msonline/set-msolcompanysettings?view=azureadps-1.0) PowerShell med parameteren **AllowAdHocSubscriptions.** Du kan finde flere oplysninger under [Hvordan styrer jeg selvbetjeningsindstillinger?](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/self-service-purchase-faq?view=o365-worldwide)

- Hvis du vil se alle tilgængelige programmer, skal du se [Tilgængelige selvbetjeningsprogrammer](https://docs.microsoft.com/microsoft-365/admin/misc/self-service-sign-up?view=o365-worldwide#available-self-service-programs).

Du kan finde fuldstændige oplysninger [under Brug af selvbetjeningstilmelding i organisationen](https://docs.microsoft.com/microsoft-365/admin/misc/self-service-sign-up?view=o365-worldwide).
