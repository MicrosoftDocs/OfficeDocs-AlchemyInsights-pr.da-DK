# <a name="security-defaults-is-enabled"></a>Sikkerhedsstandarder er aktiveret

Indstillingen for sikkerhedsstandarder beder alle brugere om multifaktorgodkendelse.

Sådan administreres sikkerhedsstandarder:

1. Log på Azure- [portalen](https://ms.portal.azure.com/)   med dine globale administratorlegitimationsoplysninger.
2. Gå til **Azure Active Directory**-  >  **Egenskaber**.
3. Klik på **Administrer sikkerhedsstandarder** nederst på siden.
4. Klik på **Ja** for at aktivere sikkerhedsstandarder eller **Nej** for at deaktivere sikkerhedsstandarder.

Hvis du vil have mere at vide, skal du se [Hvad er sikkerhedsstandarder?](https://docs.microsoft.com/azure/active-directory/fundamentals/concept-fundamentals-security-defaults).
