# <a name="manage-who-can-create-microsoft-365-groups"></a>Administrere, hvem der kan oprette Microsoft 365-grupper

Som standard kan alle i organisationen oprette O365-grupper.
  
Hvis du vil ændre det, kan du oprette en sikkerhedsgruppe med PowerShell, der begrænser gruppeoprettelsen til et undersæt af brugere.
  
Følg den trinvise vejledning her: [Administrer, hvem der kan oprette Microsoft 365-grupper](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
