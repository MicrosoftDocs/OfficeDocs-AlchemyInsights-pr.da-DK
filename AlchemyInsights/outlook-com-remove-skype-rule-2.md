# <a name="how-do-i-turn-off-skype-in-outlookcom"></a>Hvordan slår jeg Skype i Outlook.com?

Du kan ikke fjerne Skype fra Outlook.com, men du kan deaktivere beskeder:

1. Åbn **Indstillinger for** > **generelle** > **[meddelelser](https://outlook.live.com/mail/options/general/notifications)**. 
2. Deaktivere besked-typer, du ikke længere vil under **Chat og opkald**.

Yderligere oplysninger om brug af Skype læse [tale med familie og venner på Skype i Outlook.com](https://support.office.com/article/83c6a5b1-3921-479c-b9e9-e753ce59c1fa).
