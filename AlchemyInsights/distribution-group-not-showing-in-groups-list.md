# <a name="distribution-group-not-showing-in-groups-list"></a>Distributionsgruppe, der ikke vises på listen grupper

Det kan tage op til 60 minutter før den nye distributionsliste vises i Admin center.
  
Hvis du vil tilføje medlemmer eller foretage andre ændringer på nuværende tidspunkt, skal du bruge funktionen grupper i [Exchange Admin Center](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1).
