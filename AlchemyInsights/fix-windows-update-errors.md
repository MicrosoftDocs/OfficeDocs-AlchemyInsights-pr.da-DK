# <a name="fix-windows-update-errors"></a>Ret Windows Update-fejl

Hvis du får en fejlkode, mens du henter og installerer Windows Update, kan **fejlfindingsværktøjet til opdatering** hjælpe med at løse problemet.

- [Fejlfinding stiltiende af hent opdateringer til Windows 10](https://support.microsoft.com/en-us/help/4027322/windows-update-troubleshooter)
- Kør fejlfindingsværktøjet. Når det er gjort, er det bedste fremgangsmåde at genstarte enheden. Søg derefter efter nye opdateringer ved at gå til [Start > Indstillinger > Opdater & Sikkerheds> Windows Update](ms-settings:windowsupdate).

Hvis de stadig ikke virker, kan du se [Windows Update: Ofte stillede spørgsmål](https://support.microsoft.com/help/12373/windows-update-faq) for at få flere oplysninger.
