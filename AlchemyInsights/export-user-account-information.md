# <a name="export-user-account-information"></a>Eksportere kontooplysninger

Eksportere brugerkonto oplysninger, herunder oplysninger om licenser, adgangskodepolitik, blokerede konti, AAD synkroniseringsstatus og mere, gå til **brugere** > [aktive brugere](https://go.microsoft.com/fwlink/p/?linkid=834822) side og derefter vælge **Eksporter brugere**.
