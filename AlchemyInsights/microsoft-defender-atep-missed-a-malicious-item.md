# <a name="microsoft-defender-atp-missed-a-malicious-item"></a>Microsoft Defender ATP gik glip af et skadeligt element

- Send den skadelige fil til Microsoft til analyse på [Microsoft Defender Response Portal](https://www.microsoft.com/wdsi/filesubmission/). 
- Hvis du er logget på, kan du finde afsendelsesoversigten for en fil på [Microsoft Defender Response Portal](https://www.microsoft.com/wdsi/submissionhistory).
- Yderligere oplysninger finder du i [Send filer til analyse](https://docs.microsoft.com/windows/security/threat-protection/intelligence/submission-guide).
