# <a name="desktop-app-for-sharepoint"></a>Skrivebords program til SharePoint

Din organisation er angivet i det private preview-program til skrivebords programmet til SharePoint.

Microsoft understøtter i øjeblikket ikke skrivebords programmet til SharePoint (private Preview). Brug dine tilmeldte Microsoft private preview-partnere for at få hjælp til skrivebords programmet til SharePoint.
