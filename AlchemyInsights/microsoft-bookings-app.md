# <a name="microsoft-bookings-app"></a>Appen Microsoft Bookings

Microsoft Bookings er en app til aftaleplanlægning, der hjælper dig med at holde styr på dine reservationer, dine medarbejdere og dine kunder. Du kan finde flere oplysninger eller foretage fejlfinding i bookingappen i [Bookings-artikler](https://support.office.com/article/b9c9295c-c654-4b10-b5cc-f739825fc092).
