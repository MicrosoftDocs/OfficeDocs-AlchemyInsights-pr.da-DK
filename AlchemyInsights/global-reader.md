# <a name="global-reader"></a>Global læser

Brugere i denne rolle kan læse indstillinger og administrative oplysninger på tværs af Microsoft 365-tjenester, men de kan ikke udføre administrationshandlinger. Global læser er den skrivebeskyttede modstykke til global administrator.
OneDrive Administration understøtter ikke rollen Global læser.

Hvis du vil have mere at vide om Global reader og dens aktuelle begrænsninger, skal du se [Global Reader](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#global-reader).
