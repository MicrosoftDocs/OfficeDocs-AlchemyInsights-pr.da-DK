# <a name="global-reader"></a>Global læser

Brugere i denne rolle kan læseindstillinger og administrative oplysninger på tværs af Microsoft 365-tjenester, men de kan ikke udføre administrationshandlinger. Global læser er det skrivebeskyttede modstykke til den globale administrator.
OneDrive administration understøtter ikke rollen Global læser.

Du kan få mere at vide om global Reader og de aktuelle begrænsninger i [Global Reader](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#global-reader).
