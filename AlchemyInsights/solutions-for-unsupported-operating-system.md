# <a name="solutions-for-unsupported-operating-system"></a>Løsninger til "Ikke-understøttet operativsystem"



Når du forsøger at installere Office 365, kan du se en **ikke-understøttet operativsystem** fejlmeddelelse Hvis enheden ikke opfylder [systemkravene](https://products.office.com/office-system-requirements).
  
Kræver Office 365:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 eller Windows Server 2008 R2
    
    ELLER
    
- Mac OS X 10,10
    
Hvis du har fået denne meddelelse under forsøget på at installere Office på en mobil enhed eller en Chromebook, se, [hvordan du installerer og kører Microsoft Office på en Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
