# <a name="submit-powerapps-or-microsoft-flow-support-requests"></a>Sende anmodninger om PowerApps eller Flow Microsoft support

Hvis du er en **PowerApps** eller **Microsoft Flow** -kunde kan du sende os din billet via den nye [Power platform Admin Center](https://admin.powerplatform.microsoft.com/support?newTicket&product=15819). Den nye portal indeholder en mere direkte vej til assisteret support, der er optimeret til PowerApps og andre Power platform produkter.

[Send anmodning om support til Microsoft-Flow](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)
