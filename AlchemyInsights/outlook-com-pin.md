# <a name="change-your-pin"></a>Ændre din pinkode

Outlook.com bruger ikke en pinkode, men du kan logge på din PC med en pinkode eller Windows Hello. Sådan ændrer du din pinkode i Windows 10:

1. Vælg knappen Start, eller tryk på Windows-tasten på tastaturet, og vælg derefter **Indstillinger**  >  **Accounts**  >  **for konto logonindstillinger**.
2. Vælg knappen **Skift** under **fastgørelse**.
3. Angiv din nuværende pinkode, og Angiv og Bekræft din nye pinkode under.

[Læs mere om Windows Hello.](https://support.microsoft.com/help/17215/)
