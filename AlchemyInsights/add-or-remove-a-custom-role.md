# <a name="add-or-remove-a-custom-role"></a>Tilføje eller fjerne en brugerdefineret rolle

Hvis du vil administrere Azure-ressourcer, skal du se [tilføje eller fjerne Azure-rolletildelinger ved hjælp af Azure-portalen](https://docs.microsoft.com/azure/role-based-access-control/role-assignments-portal).

Hvis du vil administrere brugerdefinerede roller for Azure Active Directory (Azure AD), skal du se [brugerdefinerede administratorroller i Azure Active Directory (Preview)](https://docs.microsoft.com/azure/active-directory/users-groups-roles/roles-custom-overview).
