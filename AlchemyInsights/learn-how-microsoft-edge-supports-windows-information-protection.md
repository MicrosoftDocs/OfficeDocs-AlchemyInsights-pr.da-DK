# <a name="learn-how-microsoft-edge-supports-windows-information-protection-wip"></a>Få mere at vide om, hvordan Microsoft Edge understøtter Windows-informations beskyttelse (IGVA)

Efterhånden som antallet af Fjern arbejdere stiger, bliver antallet af personlige og arbejdsaktiviteter, der opstår på virksomhedens enheder, større. Dette øger derefter risikoen for, at virksomhedens data kan blive delt uden for arbejdspladsen. For at afhjælpe denne risiko understøtter Microsoft Edge (version 81 eller nyere) Windows information Protection (via): en Windows-10-funktion, der hjælper med at beskytte virksomhedsdata mod uautoriserede og utilsigtede lukketider.
