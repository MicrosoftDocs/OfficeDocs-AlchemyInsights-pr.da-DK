# <a name="recover-missing-or-deleted-site-collections"></a>Gendanne manglende eller slettede grupper af websteder

Se følgende artikler for at få oplysninger om, hvordan du gendanner manglende eller slettede grupper af websteder i SharePoint.

[Gendan slettede websteder](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

[Gendanne ved hjælp af SPO PowerShell-modul](https://support.office.com/article/Introduction-to-the-SharePoint-Online-Management-Shell-C16941C3-19B4-4710-8056-34C034493429)

Hvis du ikke kan finde webstedet, kan du [søge i overvågningsloggen i Microsoft 365 Security &amp; Compliance Center](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance).

**Bemærk:** "Moderne" teamwebsteder er ikke synlige i brugergrænsefladen **til Klassisk** SharePoint-administrator. Du kan få adgang til listen over "moderne" websteder fra [New Sharepoint Administration](https://docs.microsoft.com/sharepoint/get-started-new-admin-center).
