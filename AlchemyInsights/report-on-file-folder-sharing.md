# <a name="report-on-file-and-folder-sharing-in-a-sharepoint-site"></a>Rapportere om fil-og mappedeling på et SharePoint-websted

Du kan oprette en CSV-fil med detaljer om hver unik fil, bruger, tilladelse og link på et givent SharePoint-websted eller OneDrive-bibliotek. Dette kan hjælpe dig med at forstå, hvordan deling bruges, og om nogen filer eller mapper deles med gæster.

Du finder flere oplysninger [ved at rapportere om fil-og mappedeling på et SharePoint-websted](https://docs.microsoft.com/sharepoint/sharing-reports).

> [!NOTE]
> Denne funktion er i øjeblikket udrullet til første Release lejere og vil gradvist rullende ud til alle produktions lejere.
