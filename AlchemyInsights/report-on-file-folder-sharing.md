# <a name="report-on-file-and-folder-sharing-in-a-sharepoint-site"></a>Rapport om fil- og mappedeling på et SharePoint-websted

Du kan oprette en CSV-fil med detaljer om alle entydige filer, brugere, tilladelser og links på et givet SharePoint-websted eller OneDrive-bibliotek. Dette kan hjælpe dig med at forstå, hvordan deling bruges, og om filer eller mapper deles med gæster.

Rapportér [om fil- og mappedeling på et SharePoint-websted](https://docs.microsoft.com/sharepoint/sharing-reports).

> [!NOTE]
> Denne funktion er i øjeblikket installeret på first release-lejere og vil gradvist blive udrullet til alle produktionslejere.
