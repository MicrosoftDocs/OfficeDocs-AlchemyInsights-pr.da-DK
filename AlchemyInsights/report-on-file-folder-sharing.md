# <a name="report-on-file-and-folder-sharing-in-sharepoint-sites"></a>Rapport om deling af filer og mapper på SharePoint-websteder

Du kan oprette en CSV-fil, der beskriver alle entydige filer, brugere, tilladelser og links på et bestemt SharePoint-websted eller OneDrive-bibliotek. Det kan hjælpe dig med at forstå, hvordan deling bruges, og om filer eller mapper deles med gæster.

Hvis du vil have mere at vide, skal du [rapportere om fil-og mappedeling på et SharePoint-websted](https://docs.microsoft.com/sharepoint/sharing-reports).

> [!NOTE]
> Denne funktion er i øjeblikket installeret på First Release-lejere og udrulles gradvist til alle produktions lejere.
