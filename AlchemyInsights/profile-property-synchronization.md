# <a name="profile-property-synchronization"></a>Egenskaben synkronisering af brugerprofil

Det er ikke muligt at knytte flere egenskaber for AD Import skal synkroniseres fra Azure Active Directory til brugerprofilprogrammet. AD Import er begrænset til en række forudkonfigurerede egenskaber for at sikre ensartet ydeevne af timerjobbet.
  
[Yderligere oplysninger om de profilegenskaber, der synkroniseres](https://go.microsoft.com/fwlink/?linkid=875671)
