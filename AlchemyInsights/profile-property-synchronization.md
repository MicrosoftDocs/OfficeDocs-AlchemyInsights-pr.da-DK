# <a name="profile-property-synchronization"></a>Synkronisering af profilegenskaber

Det er ikke muligt at tilknytte yderligere egenskaber for AD-import for at synkronisere fra Azure Active Directory til brugerprofilprogrammet. AD-import er begrænset til et forudkonfigureret sæt egenskaber for at sikre ensartet ydeevne for timerjobbet.
  
[Få mere at vide om de profilegenskaber, der synkroniseres](https://go.microsoft.com/fwlink/?linkid=875671)
