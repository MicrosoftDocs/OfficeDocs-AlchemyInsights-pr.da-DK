# <a name="perform-access-reviews-in-azure-active-directory"></a>Udføre adgangs gennemganger i Azure Active Directory

Udvalgte korrekturlæsere kan udføre Azure Active Directory-adgang til gennemsyn af gruppemedlemskaber, programadgang og rolletildelinger. Disse anmeldelser hjælper med at reducere de sikkerhedsrisici, organisationen kan komme ud for, når brugere har flere adgang, end de har brug for. Du kan udføre adgangs gennemganger ved hjælp af [mine apps](https://go.microsoft.com/fwlink/?linkid=2134605) eller [min adgang](https://go.microsoft.com/fwlink/?linkid=2134505).
