# <a name="check-aad-identity-sync-status"></a>Kontrollér AAD-identitets synkroniseringsstatus

Det er nemt at kontrollere status for synkronisering:
  
- Log på Microsoft 365 administration, og vælg **DirSync-status** på startsiden.

- Alternativt kan du gå til brugere, der er \> aktive brugere, og på siden aktive brugere vælge flere \> Katalogsynkronisering.

- I ruden Katalogsynkronisering skal du vælge gå til administration af DirSync.

Hvis der er problemer med din Katalogsynkronisering, vises fejlene på denne side. Du kan finde flere oplysninger om forskellige fejl, du kan støde på, i [vise katalog synkroniseringsfejl](https://docs.microsoft.com//office365/enterprise/identify-directory-synchronization-errors).
