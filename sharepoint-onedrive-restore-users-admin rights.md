<p><span data-ttu-id="71b8a-103"><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Dette problem opstår oftest, når en bruger slettes og genoprettes med det samme brugerens hovednavn (UPN). A second scenario involves directory synchronization with an Active Directory organizational unit (OU).
